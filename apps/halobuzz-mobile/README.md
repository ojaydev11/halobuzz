# HaloBuzz Mobile App - Revolutionary Social Streaming Platform

## 🚀 Overview

HaloBuzz is a next-generation social streaming platform that combines live streaming, gaming, AI content creation, NFT marketplace, and creator economy into one addictive mobile experience. Built to dominate the industry for the next 10-20 years with cutting-edge features and unparalleled user engagement.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Expo CLI
- iOS Simulator or Android Emulator
- Backend API running

### Installation
```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

### Environment Setup
Create `.env` file with:
```
API_BASE_URL=your_backend_url
AGORA_APP_ID=your_agora_id
AGORA_APP_CERTIFICATE=your_agora_cert
```
